IGB100

So for the very first game creation unit, as a team with 2 others (a business student and another software tech). We created our very first game together which was "Look No hands" using Unity and C#. It was based on a bike that goes down hill and picks up speed and the player has to dodge objects. The objects would RNG each time the game would play from a raycast, it was a simple game and was ok...

Our second game we created together was "Wobble", it was based off a game show that my team mate Cameron came up with. The user had to balance on a platform and throw objects to match the shape and colour for additional points. For a simple game with simple models, I found this project quite enjoyable and the result was alright
