IGB100

So for the very first game creation unit, as a team with 2 others (a business student and another software tech). We created our very first game together which was "Look No hands" using Unity and C#. It was based on a bike that goes down hill and picks up speed and the player has to dodge objects. The objects would RNG each time the game would play from a raycast, it was a simple game and was ok...

Our second game we created together was "Wobble", it was based off a game show that my team mate Cameron came up with. The user had to balance on a platform and throw objects to match the shape and colour for additional points. For a simple game with simple models, I found this project quite enjoyable and the result was alright

In this unit we had to create a portfolio and also make a project that would help contriubte to it. I choose to create this website to use to show my work to potential employers. Using previous knowledge of HTML, CSS and GitHub Pages I was able to create this website, while also learning and picking up new skills as well. This website will be something I will continue to add to, as I proceed in my programming carrer

The project I choose to create was multiplayer FPS shooter using networking (Photon Pun2) and C# in Unity with also a toggleable aimbot.

I really wanted to push myself to get a somewhat decent result and even though I was told by many people that I may of overscoped this project, I continued to proceed with the project anyway. As of 4/11/2020 when I did complete the project, the documentation of PhotonPun2 was changing heaps since it was new and a lot of the documentation was also out of date.
